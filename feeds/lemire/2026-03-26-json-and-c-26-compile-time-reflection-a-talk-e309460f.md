# JSON and C++26 compile-time reflection: a talk

![](https://lemire.me/blog/wp-content/uploads/2026/03/Capture-decran-le-2026-03-25-a-20.36.07-150x150.png)

The next C++ standard (C++26) is getting exciting new features. [One of these features is compile-time reflection](https://lemire.me/blog/2025/06/22/c26-will-include-compile-time-reflection-why-should-you-care/). It is ideally suited to serialize and deserialize data at high speed. To test it out, we extended our fast JSON library ( [simdjson](https://github.com/simdjson/simdjson)) and we gave a talk at CppCon 2025. The video is out on YouTube.

[Our slides are also available](https://simdjson.github.io/simdjson_talks/cppcon2025/cppcon_2025_slides.html).
