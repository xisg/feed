# Podman 6.0 released

Version 6.0.0 of the [Podman](https://podman.io/) container-management tool has been released. Notable new features include the ability to set multiple static IP addresses for containers, improvements in network isolation that make Podman more compatible with Docker, changes to the way [Quadlet](https://docs.podman.io/en/latest/markdown/podman-quadlet.1.html) commands function, many new options for many existing `podman` commands, and [a
rewrite of Podman's configuration file handling](https://blog.podman.io/2026/06/podman-6-configuration-file-changes/). There are many breaking changes; see the [release
notes](https://github.com/podman-container-tools/podman/releases/tag/v6.0.0) for a full list of all new features, changes, and bug fixes.
