# Homebrew 6.0.0 released

[Version\
6.0.0](https://brew.sh/2026/06/11/homebrew-6.0.0/) of the [Homebrew](https://brew.sh/)
package-management system has been released. Notable changes in this
release include the introduction of [tap trust](https://docs.brew.sh/Tap-Trust) to improve
supply-chain security, improvements in sandboxing on Linux, a number
of performance tweaks, and many other changes.

See the [changelog](https://github.com/Homebrew/brew/releases/tag/6.0.0)
for a full list. LWN [covered](https://lwn.net/Articles/1046236/) Homebrew in
November 2025.
