# LLVM Developer Meeting 2012

The [2012 LLVM Developer Meeting](http://llvm.org/devmtg/2012-11/) is now complete and was a great success. If you are interested in the talks, a complete set of [slides, videos, and posters](http://llvm.org/devmtg/2012-11/) are now available. The LLVM Developer Meeting would not be possible without support from our sponsors. Many thanks to Apple, QuIC, Google, and Intel. -Chris
