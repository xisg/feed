# Announcing the program for the 2019 LLVM Developers' Meeting - Bay Area

Announcing the program for the [2019 LLVM Developers' Meeting](http://llvm.org/devmtg/2019-10/) in San Jose, CA! This program is the largest we have ever had and has over 11 tutorials, 29 technical talks, 24 lightning talks, 2 panels, 3 birds of a feather, 14 posters, and 4 SRC talks. Be sure to [register](https://www.eventbrite.com/e/2019-llvm-developers-meeting-bay-area-tickets-63481303287) to attend this event and hear some of these great talks.
