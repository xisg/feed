# LLVM receives first ever SIGPLAN Software Systems Award

Today the [ACM announced](http://www.acm.org/press-room/news-releases/2010/sigplan-software-award/view) that Chris Lattner and the LLVM Project are the recipient of the [Programming Languages Software Award](http://www.sigplan.org/award-software.htm).

This award recognizes the impact that LLVM has had on the compiler research community, which can be seen in the large number of [research publications using LLVM](http://llvm.org/pubs/) and the number of compiler courses that are using LLVM to teach compiler construction and optimization techniques. As part of this award, the ACM is donating $2500 to the LLVM project.
